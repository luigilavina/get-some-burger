# Get Some Burger 🍔

Sistema fullstack para hamburgueria com autenticação de usuários, painel administrativo e gerenciamento dinâmico de produtos.

## Funcionalidades

- Cadastro e login de usuários
- Login de administrador
- Painel administrativo
- CRUD de produtos
- Menu dinâmico integrado ao banco de dados
- Categorias de produtos
- Exibição de imagens dos produtos

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Git/GitHub
- Insomnia

## Como executar o projeto

### Backend

```bash
cd backend
npm install
npm start
```

Crie um arquivo `.env` dentro da pasta `backend`:

```env
MONGO_URI=sua_string_do_mongodb
PORT=3000
```

### Frontend

Abra o arquivo:

```bash
frontend/index.html
```

ou utilize a extensão Live Server no VS Code.
